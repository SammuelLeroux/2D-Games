# 2D GAMES

Projet Epitech:\
Creation d'un jeu en 2D en utilisant le language orienté objet Java.

# Demarrage
- Cloner le projet
git clone https://github.com/SammuelLeroux/2D-Games.git

# Lancer le jeu
Récupérer le .jar et l'executer:
    - manuellement
    - via ligne de commande : java -jar 2D-Games.jar
